部分项目：

  • NeurolingsCE
    跨平台桌面看板娘（Shimeji）应用，基于 Shijima-Qt 进行深度修改，使用 C++17 / Qt6 构建，支持 Windows、Linux 和 macOS。项目支持看板娘资源包导入、拖拽交互、多显示器、自定义缩放、HTTP REST API 与多语言界面，可用于桌面交互、角色陪伴和社区定制化场景。

  • neuroSangSpider
    基于 Python 开发的 Neuro-sama / Evil 歌曲相关工具，主要用于从 Bilibili 获取歌回视频内容，并支持歌曲搜索、指定歌回下载、本地播放、基础推荐和 GUI 可视化交互。项目面向 Neuro 社区的歌曲整理、播放与存档需求，体现了爬虫、媒体处理和桌面工具开发能力。

  • Simple-TTS-Workflow
    一个简单的本地 TTS Web 工作流项目，默认使用 VoxCPM2 进行可控声音克隆，同时保留 Qwen3-TTS 的 Clone 与 VoiceDesign 流程。项目支持多行文本批量生成音频，并将生成结果保存为 .wav 文件，适合用于本地语音合成、角色语音实验和轻量化 TTS 工作流搭建。

  • ArrowFight
    一个基于 Java 的 Arrow Fight Plugin 项目，属于 Minecraft / Bukkit / Paper 插件方向的实践项目。项目围绕游戏机制扩展与插件开发展开，可体现 Java 工程开发、Minecraft 服务端插件设计以及游戏玩法逻辑实现能力。

  • LiteVoiceTrie
    一个轻量级中文语音指令匹配器，使用 Python 编写，面向智能家居、车机、语音助手等场景，将 ASR 识别文本映射到合法指令词典。项目支持精确匹配、一次编辑容错的模糊匹配、ASR 文本归一化、末尾语气词剥离，并附带测试数据生成与自动化评测脚本，适合作为嵌入式语音命令匹配方向的原型模板。

  • MorseFold
    一个结构化 Morse 编码压缩与基准测试项目，探索如何在保持完全可逆的前提下，对 Morse 符号序列进行轻量化结构重编码。项目通过连续等长分组、符号偏置和交替模式检测等规则，将标准 Morse 编码转换为更短的结构化表示，适合用于算法原型验证、符号序列压缩实验和可逆编码研究。

  • AsianDifficulty
    一个 Minecraft AsianDifficulty MOD 项目，围绕“亚洲难度”这一创意玩法对 Minecraft 生存体验进行扩展。模组加入了多种特殊死亡或惩罚机制，例如怕黑、恐高、感冒、一氧化碳中毒、高处生寒、过重、劳累过度、被僵尸感染等，使游戏过程更具整活性和挑战性。

  • C_SimpleToolBox
    一个面向 C 语言初学者的 Windows 控制台工具箱项目，用于帮助新手理解较完整的 C 语言项目结构。项目包含选择排序、二分查找、冒泡排序等算法演示功能，也包含英文诗藏头解析、密码生成器等实用工具，适合作为 C 语言大程序结构、模块拆分和菜单式控制台程序设计的入门示例。
