清空缓存区

连续输入时要记得清空缓存区,否则就会出现奇怪的bug

  1. 使用scanf()

    scanf("%*[^\n]"); 
scanf("%*c");

  2. 使用while()

    while((ch = getchar()) != ‘\n’ && ch != EOF);

    万能语句

  3. 使用fflush()

    fflush(stdin);
